# Google Cybersecurity Professional Certificate — Lab Portfolio

A hands-on portfolio documenting my work through the **Google Cybersecurity Professional Certificate** (Coursera, 2026), funded by **Michigan Works! WIOA** under the Individual Training Account program.

This repo captures lab exercises, project writeups, tool exploration, and lessons learned across the 8-course program — designed to demonstrate practical, employer-ready cybersecurity skills.

## 🎯 Program Overview

The Google Cybersecurity Certificate is an 8-course program covering foundational to applied cybersecurity skills, with 170+ hours of hands-on instruction. Topics include threat identification, risk management, network security, SIEM tools, Linux, Python automation, and incident response.

**Prepares for:** CompTIA Security+ certification (planned next step)

## 🛠️ Tools & Technologies

- **SIEM:** Splunk, Google Chronicle
- **Network Analysis:** Wireshark, tcpdump
- **Intrusion Detection:** Suricata
- **Operating Systems:** Linux (command line, permissions, shell)
- **Scripting:** Python, SQL
- **Frameworks:** NIST CSF, CIA Triad, OWASP
